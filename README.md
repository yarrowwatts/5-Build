## 5-Build | Independent Film Community Site

## Project Description
This project for GIT 215 (Summer A 2025 at ASU) is a responsive, multi-page website for an fictional independent film and creative arts community. 

The website uses semantic HTML, images with descriptive alt text, and mobile-first CSS. The layout begins as a single-column design on smaller screens and changes at larger screen sizes using a media query.

## Pages Included

- Home
- Film Screenings
- Workshops
- Collaborations
